Welcome to the Customer Solutions Onboarding Repository!

This repository contains boilerplate & solution code for training modules delivered by the Customer Solutions team at Algolia.

## Repository Structure

- `<module>/<tech stack>/boilerplate`: Contains the starting code for each module in various tech stacks.
- `<module>/<tech stack>/solution`: Contains the completed solution code for each module for the specific tech stack.

## Getting Started

The code contains all necessary elements to be forked in codesandbox.io the go to tool used by the Customer Solutions team for training sessions. Refer to the specific module README files for instructions on how to set up and run each module.
